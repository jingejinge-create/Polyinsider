Real-time insider activity tracker for Polymarket prediction markets.

## Features

- **Live Data**: Fetches real trades from Polymarket API
- **6-Metric Scoring**: Advanced insider detection algorithm
- **Filterable Dashboard**: Filter by size, score, time
- **Score Breakdown**: See why trades look suspicious

## Deploy to Vercel

1. Upload all files to a new GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Import the repository
4. Click Deploy

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Polymarket Data API

## Disclaimer

This tool identifies patterns that MAY indicate insider activity. High scores do not guarantee insider trading.
